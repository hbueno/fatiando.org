Instalando o Fatiando a Terra

Windows

No momento instalação no Windows está, digamos, antipática. Entre em contato conosco para receber instruções e ajuda sobre como realizar essa façanha.

Linux

A maioria das distribuições de GNU/Linux já vem com Python 2.7 instalado. Em seguida, instale as seguintes bibliotecas utilizando seu gerenciador de pacotes (Synaptic, Ubuntu Software Center, etc):

  • python-numpy
  • python-scipy
  • python-matplotlib
  • python-imaging
  • python-pip
  • python-dev

Baixe o Fatiando a Terra em formato .tar.gz. Em seguida digite no terminal:

sudo pip install fatiando-0.0.1.tar.gz

Para testar a instalação, rode um dos exercícios práticos (arquivos .py) no terminal:

python exercicio1.py

Todos os exercícios práticos devem abrir uma janela com um gráfico. Se isto não acontecer, entre em contato.
